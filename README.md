# Smart Class Performance Analytics

A web-based class analytics dashboard powered by Python (Brython). This tool helps teachers manage student marks, calculate grades, and analyze class performance in real-time.

## Features
- **Dynamic Curriculum:** Add any number of subjects with specific total marks.
- **Auto-Grading:** Instant calculation of percentages and grades.
- **Analytics Dashboard:**
    - Displays Top 3 students.
    - Identifies lowest scoring student.
    - Categorizes students based on class average.
- **Interactive UI:** Responsive design with dynamic input generation.

## How to Run
1. Clone this repository.
2. Open `index.html` in your browser (use Live Server in VS Code for local development).
3. Start adding subjects in the "Manage Subjects" section.
4. Input student names and marks to see real-time analytics.

## Technologies Used
- **Python (Brython):** To run Python code directly in the browser.
- **HTML5 & CSS3:** For structuring and styling the dashboard.
- **JavaScript:** For base browser interaction.
